Monitoramento de Consumo Sustentável

O módulo de monitoramento é a espinha dorsal da sustentabilidade no Nexus. Ele utiliza sensores IoT para coletar dados em tempo real sobre o consumo de água e energia de cada unidade. Essa informação é processada e armazenada em um banco de dados **MongoDB**.

Quando o consumo excede um limite pré-definido, o sistema **dispara alertas automáticos** para os moradores e síndicos, promovendo a conscientização e a economia de recursos.

Gerenciamento de Ocorrências

Este módulo simplifica a comunicação entre moradores e a administração do condomínio. Um morador pode registrar uma ocorrência (como um problema no elevador ou vazamento de água) que é enviada diretamente para o síndico.

Cada ocorrência tem um status que evolui conforme o problema é tratado. Inicialmente, o status é **"Pendente"**. Após ser analisada pelo síndico, a ocorrência pode se tornar um comunicado ou ser resolvida.

Segurança com Integração IoT

O Nexus eleva a segurança do condomínio ao integrar o controle de acesso com dispositivos IoT. O sistema permite o cadastro e a gestão de visitantes de forma segura, usando um banco de dados **PostgreSQL** para garantir a integridade dos dados, conforme exigido pela LGPD.

A autenticação é reforçada por tecnologias como **JWT (JSON Web Tokens)**, que garantem que apenas usuários autorizados tenham acesso às informações. Essa arquitetura de segurança é projetada para ser robusta e escalável, protegendo tanto os dados quanto os moradores.
